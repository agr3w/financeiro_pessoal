# Finance Favicon Design

Este projeto tem como objetivo desenvolver um favicon em formato SVG para uma aplicação de gerenciamento financeiro pessoal. O favicon é um elemento importante para a identidade visual da aplicação, ajudando a criar uma conexão instantânea com os usuários.

## Estrutura do Projeto

O projeto é organizado da seguinte forma:

- **src/assets/reference-logos**: Contém logotipos de referência que podem ser utilizados como inspiração ou base para o design do favicon.
  
- **src/components**: Inclui ícones SVG que podem ser utilizados no design do favicon:
  - `DollarSign.svg`: Ícone de cifrão, representando finanças.
  - `ChartIcon.svg`: Ícone de gráfico, simbolizando análises financeiras.
  - `WalletIcon.svg`: Ícone de carteira, associado ao gerenciamento de finanças pessoais.

- **src/drafts**: Contém versões em desenvolvimento do favicon:
  - `favicon-v1.svg`: Primeira versão do favicon.
  - `favicon-v2.svg`: Segunda versão com melhorias.
  - `favicon-v3.svg`: Terceira versão com ajustes finais.

- **src/final**: Contém as versões finais do favicon:
  - `favicon.svg`: Versão final do favicon.
  - `favicon-dark-mode.svg`: Versão otimizada para modo escuro.

- **src/tools/viewer.html**: Ferramenta de visualização para ver os favicons em diferentes contextos ou tamanhos.

## Instruções de Uso

1. **Visualização**: Utilize o arquivo `viewer.html` para visualizar os favicons em diferentes tamanhos e contextos.
2. **Desenvolvimento**: As versões em `drafts` podem ser editadas e melhoradas conforme necessário.
3. **Implementação**: Utilize os arquivos em `final` para implementar o favicon na sua aplicação.
